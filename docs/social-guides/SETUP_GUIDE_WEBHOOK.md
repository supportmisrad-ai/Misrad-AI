## 📘 שלב 1: הרשמה ל-Make

### אם אין לך חשבון:
```
1. היכנס ל: https://www.make.com
2. לחץ "Sign Up" (הרשמה)
3. השתמש ב-Google או Email
4. אשר את המייל
```

### אם יש לך חשבון:
```
1. היכנס ל: https://www.make.com
2. התחבר עם הסיסמה שלך
```

---

## 🎯 שלב 2: יצירת Scenario חדש

### 2.1 צור Scenario
```
1. לחץ על "Create a new scenario"
2. תן לו שם: "Misrad AI → Social Media"
3. לחץ "Continue"
```

### 2.2 הוסף Webhook Trigger
```
1. חפש "Webhooks" במודולים
2. בחר "Webhooks → Custom webhook"
3. לחץ "Create a webhook"
4. תן לו שם: "Social Post"
5. לחץ "Save"
```

### 2.3 העתק את ה-URL
```
Make יראה לך URL ארוך, משהו כמו:
https://hook.make.com/abc123xyz456...

→ לחץ על "Copy to clipboard"
→ שמור אותו - נצטרך בשלב הבא!
```

---

## 🔌 שלב 3: חיבור ל-Misrad AI

### 3.1 היכנס למערכת
```
1. היכנס ל-Misrad AI
2. לך ל: הגדרות → אינטגרציות
3. לחץ על "הוסף Webhook חדש"
```

### 3.2 הדבק את ה-URL
```
1. בשדה "Webhook URL" - הדבק את ה-URL שהעתקת מ-Make
2. בחר "Make" בתפריט
3. סמן: ☑ "פרסום פוסט"
4. לחץ "שמור"
```

### ✅ כעת Misrad AI מחובר ל-Make!

---

## 📱 שלב 4: הוסף פלטפורמות (Facebook לדוגמה)

### 4.1 חזור ל-Make
```
1. ב-Scenario שיצרת
2. לחץ על "+" אחרי ה-Webhook
3. חפש "Facebook"
4. בחר "Facebook → Create a post"
```

### 4.2 התחבר לפייסבוק
```
1. לחץ "Add connection"
2. התחבר עם חשבון הפייסבוק שלך
3. אשר הרשאות
4. בחר את הדף שלך
```

### 4.3 קבע את התוכן
```
בשדה "Message" (התוכן):
→ לחץ על השדה
→ מהמפה (Map), בחר: content
→ זה ימפה את תוכן הפוסט אוטומטית
```

### 4.4 הוסף תמונה (אופציונלי)
```
בשדה "Image URL":
→ בחר מהמפה: mediaUrl
→ זה יוסיף תמונות אם יש
```

## 🔄 שלב 5: הוסף עוד פלטפורמות

### Instagram
```
1. לחץ על "+" אחרי הפייסבוק
2. חפש "Instagram"
3. בחר "Instagram → Create a post"
4. חזור על אותו תהליך
```

### LinkedIn
```
1. לחץ על "+" 
2. חפש "LinkedIn"
3. בחר "LinkedIn → Share an update"
4. חזור על אותו תהליך
```

**טיפ:** כל הפלטפורמות במקביל! Make יפרסם לכולן ביחד.

## ✅ שלב 6: הפעלה ובדיקה

### 6.1 הפעל את ה-Scenario
```
1. למעלה מימין, לחץ על המתג
2. צריך להיות "ON" (ירוק)
3. לחץ "Save"
```

### 6.2 צור פוסט בדיקה
```
1. חזור ל-Misrad AI
2. Social → פוסטים חדשים
3. כתוב: "בדיקה מ-Make ✨"
4. בחר פלטפורמות
5. לחץ "פרסם עכשיו"
```

### 6.3 בדוק ב-Make
```
1. חזור ל-Make
2. תראה שהפוסט התקבל (בצבע ירוק)
3. תראה שהוא פורסם לפייסבוק/אינסטגרם
```

### 6.4 בדוק ברשתות
```
תוך 1-2 דקות:
✅ הפוסט אמור להופיע בפייסבוק
✅ הפוסט אמור להופיע באינסטגרם
✅ הפוסט אמור להופיע בלינקדאין
```

## ❓ שאלות נפוצות

### כמה זה עולה?
- **Make חינם:** עד 1,000 פעולות/חודש
- **Zapier חינם:** עד 100 פעולות/חודש

**דוגמה:**
- 100 פוסטים × 3 פלטפורמות = 300 פעולות
- מכוסה ב-Make חינם!

### האם צריך Scenario לכל לקוח?
**לא!** Scenario אחד יכול לשרת את כולם.

אבל אם אתה סוכנות ורוצה הפרדה:
- ✅ אפשר ליצור Scenario לכל לקוח
- ✅ כל לקוח עם Webhook URL משלו

### מה קורה אם Make נופל?
המערכת תנסה שוב אוטומטית 3 פעמים.
אם זה לא עובד - תקבל התראה.

### האם אפשר להוסיף אוטומציות?
**כן!** זאת היתרון של Make.

דוגמאות:
- ✅ שמור פוסט בגוגל דרייב
- ✅ שלח התראה בטלגרם
- ✅ עדכן גיליון אקסל
- ✅ שלח למייל

## 🆘 פתרון בעיות

### הפוסט לא מתפרסם?
```
1. בדוק ב-Make History - האם התקבל?
   ❌ לא? → בדוק Webhook URL ב-Misrad AI
   ✅ כן? → בדוק חיבור לפייסבוק ב-Make

2. בדוק שה-Scenario ON (ירוק)

3. בדוק שיש לך הרשאות בדף הפייסבוק
```

### "Invalid token" ב-Make?
```
1. לך ל-Make → Connections
2. מחק את החיבור הישן לפייסבוק
3. צור חיבור חדש
4. הרשה שוב
```

### "Too many requests"?
```
הגעת למכסה של Make (1,000/חודש).
→ שדרג ל-Make Pro (9$/חודש, 10,000 פעולות)
```
