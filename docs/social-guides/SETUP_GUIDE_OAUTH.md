# 🔐 מדריך הגדרת פרסום ישיר (OAuth)

## למה כדאי לך?

**פרסום ישיר = מהיר יותר, אמין יותר**
- ✅ פרסום מיידי לרשתות החברתיות
- ✅ ללא תלות בכלים חיצוניים (Make/Zapier)
- ✅ אמין יותר (פחות כשלים)
- ✅ חינם (אין צורך במנוי Make/Zapier)

**למי זה מתאים?**
- סוכנויות שיווק
- עסקים עם פרסום תכוף
- צוותי שיווק מקצועיים

## ⏱️ זמן הגדרה: 15-20 דקות

## 📘 שלב 1: חיבור Facebook + Instagram

### מה תצטרך:
- חשבון Facebook Business Manager
- דף עסקי בפייסבוק
- חשבון Instagram עסקי (מקושר לדף)

### הוראות צעד-אחר-צעד:

#### 1.1 היכנס למערכת
```
1. היכנס ל-Misrad AI
2. לך ל: הגדרות → רשתות חברתיות
3. לחץ על "חבר Facebook"
```

#### 1.2 אשר הרשאות
```
Facebook יבקש ממך:
✅ גישה לדפים שלך
✅ גישה לפרסום פוסטים
✅ גישה לחשבון Instagram העסקי

→ לחץ "אשר" על הכל
```

#### 1.3 בחר דף
```
1. תראה רשימת כל הדפים שלך
2. בחר את הדף הרלוונטי
3. אם יש לך Instagram עסקי מקושר - הוא יתווסף אוטומטית
4. לחץ "סיום"
```

### ✅ סיימת! פייסבוק + אינסטגרם מחוברים

## 💼 שלב 2: חיבור LinkedIn

### מה תצטרך:
- חשבון LinkedIn אישי או עמוד חברה

### הוראות צעד-אחר-צעד:

#### 2.1 היכנס למערכת
```
1. היכנס ל-Misrad AI
2. לך ל: הגדרות → רשתות חברתיות
3. לחץ על "חבר LinkedIn"
```

#### 2.2 אשר הרשאות
```
LinkedIn יבקש:
✅ גישה לפרופיל שלך
✅ גישה לפרסום פוסטים
✅ גישה לעמוד החברה (אם רלוונטי)

→ לחץ "אשר"
```

#### 2.3 בחר היכן לפרסם
```
תוכל לבחור:
○ פרופיל אישי
○ עמוד חברה (אם אתה מנהל)

→ בחר ולחץ "סיום"
```

### ✅ סיימת! LinkedIn מחובר

## 🎯 שלב 3: בדיקה

### איך לבדוק שהכל עובד?

#### 3.1 צור פוסט בדיקה
```
1. לך ל: Social → פוסטים חדשים
2. כתוב משהו פשוט: "בדיקה ✨"
3. בחר את הפלטפורמות שחיברת
4. לחץ "פרסם עכשיו"
```

#### 3.2 בדוק ברשתות
```
תוך 1-2 דקות:
✅ הפוסט אמור להופיע בפייסבוק
✅ הפוסט אמור להופיע באינסטגרם
✅ הפוסט אמור להופיע בלינקדאין
```

### אם משהו לא עובד:
📞 צור קשר לתמיכה: support@misrad-ai.com

## ❓ שאלות נפוצות

### כמה זמן התחברות תקפה?
- **Facebook/Instagram:** 60 יום (נרענן אוטומטית)
- **LinkedIn:** 60 יום (נרענן אוטומטית)

אם Token פג תוקף - תקבל התראה והמערכת תבקש ממך להתחבר שוב.

### האם צריך לחבר כל לקוח בנפרד?
**כן!** כל לקוח = חיבור משלו.

אם אתה סוכנות עם 10 לקוחות:
- כל לקוח מתחבר עם חשבון הפייסבוק **שלו**
- כל לקוח מתחבר עם חשבון הלינקדאין **שלו**

### האם זה בטוח?
**מאוד!**
- ✅ הטוקנים מוצפנים
- ✅ אנחנו לא רואים סיסמאות
- ✅ רק הרשאות לפרסום (לא מחיקה)
- ✅ אפשר לנתק בכל רגע

### כמה זה עולה?
**חינם!** כלול במנוי שלך.
