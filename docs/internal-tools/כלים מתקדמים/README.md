# כלים מתקדמים

## יצירת PDF עם צילומי מסך של כל מסכי המערכת

הסקריפט מצלם את כל המסכים בפועל ויוצר PDF עם הצילומים - מושלם להראות את העיצוב והממשק לאנשים ללא גישה למערכת.

### דרישות

1. **Python packages:**
   ```bash
   pip install playwright reportlab Pillow
   ```

2. **Playwright browser:**
   ```bash
   playwright install chromium
   ```

### שימוש

1. **התחל את השרת המקומי:**
   ```bash
   npm run dev
   ```
   (השרת יעבוד על http://localhost:4000)

2. **הרץ את הסקריפט:**
   ```bash
   python "כלים מתקדמים\create_pdf.py"
   ```

   או עם URL מותאם:
   ```bash
   python "כלים מתקדמים\create_pdf.py" http://localhost:3000
   ```

3. **התחבר למערכת:**
   - הסקריפט יפתח דפדפן
   - אם המערכת דורשת התחברות, התחבר כרגיל
   - לחץ Enter בקונסול לאחר שהתחברת
   - **טיפ:** הסקריפט שומר את ה-cookies שלך, כך שבפעם הבאה לא תצטרך להתחבר שוב!

4. **הסקריפט ימשיך אוטומטית:**
   - יעבור על כל המסכים
   - יצלם אותם
   - ייצור PDF עם כל הצילומים

### פלט

- **תיקיית screenshots** - כל צילומי המסך נשמרים בתיקייה `כלים מתקדמים/screenshots/`
- **קובץ PDF** - `מסכי_המערכת.pdf` בתיקיית הפרויקט

### מה נכלל ב-PDF?

- **מסכים ראשיים** - כל המסכים הראשיים (לוח בקרה, משימות, יומן, לקוחות, וכו')
- **הגדרות** - כל תפריטי המשנה בהגדרות
- כל מסך כולל כותרת וצילום מסך מלא

### הערות

- הסקריפט משתמש ב-Playwright כדי לפתוח דפדפן ולצלם
- אם יש מסכים שדורשים הרשאות מיוחדות, ודא שיש לך גישה אליהם
- זמן הריצה תלוי במספר המסכים ובמהירות הטעינה
- אם יש שגיאה במסך מסוים, הסקריפט יפסיק ויעבור למסך הבא
