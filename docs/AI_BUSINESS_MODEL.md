## 🔑 איך המערכת יודעת לחייב כל פעולה?

### **הקוד שעושה את זה:**

```typescript
// lib/services/ai/AIService.ts

class AIService {
  async chat(params) {
    // 1. אומדן עלות (לפי מודל)
    const estimatedCost = this.estimateCost({
      model: "gpt-4-turbo",
      estimatedTokens: 500
    });
    // → 50 cents
    
    // 2. הזמן קרדיטים מהלקוח
    await this.reserveCredits({
      organizationId: params.organizationId,
      reserveCents: estimatedCost
    });
    // SQL: UPDATE organizations 
    //      SET ai_credits_balance_cents -= 50
    
    // 3. קרא ל-OpenAI
    const response = await openai.chat.completions.create({
      model: "gpt-4-turbo",
      messages: params.messages
    });
    
    // 4. חישוב עלות אמיתית (מה ש-OpenAI מחייב אותך)
    const actualTokens = response.usage.total_tokens;  // 350
    const actualCostToYou = actualTokens * 0.03 / 1000;  // $0.0105
    
    // 5. החלטה כמה לחייב את הלקוח (לפי המדיניות שלך)
    const chargedCents = Math.ceil(actualTokens / 10);  // 35 cents
    
    // 6. התאם יתרה (החזר הפרש)
    const refund = estimatedCost - chargedCents;  // 50 - 35 = 15
    await this.adjustCredits({
      organizationId: params.organizationId,
      deltaCents: refund  // +15 cents
    });
    
    // 7. לוג לצורכי מעקב
    await prisma.ai_usage_logs.create({
      data: {
        organization_id: params.organizationId,
        provider: "openai",
        model: "gpt-4-turbo",
        charged_cents: chargedCents,  // 35
        actual_cost_usd: actualCostToYou,  // $0.0105
        tokens_used: actualTokens,  // 350
      }
    });
    
    return response;
  }
}
```

## 🔍 כפתור "אינדוקס" - מה הוא עושה ולמה?

### **מה זה אינדוקס (Indexing)?**

**הגדרה פשוטה:**
תהליך שלוקח את **כל המסמכים והנתונים** בארגון ויוצר מהם **"טביעת אצבע דיגיטלית"** (Embeddings).

### **למה צריך את זה?**

**בלי אינדוקס:**
```
משתמש: "מצא לי לידים דומים ללקוח ABC"
AI: ❌ "אין לי מושג, אין לי גישה לנתונים"
```

**עם אינדוקס:**
```
משתמש: "מצא לי לידים דומים ללקוח ABC"
AI: ✅ "מצאתי 3 לידים:
     1. חברת XYZ (התעשייה זהה, גודל דומה)
     2. ארגון DEF (אותה בעיה, תקציב דומה)
     3. חברת GHI (איזור גיאוגרפי דומה)"
```
