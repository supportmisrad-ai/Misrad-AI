## 🎯 הנחות יסוד

### על המוצר
- **מחיר ממוצע:** ₪349/חודש (חבילה ממוצעת)
- **LTV (Lifetime Value):** ₪349 × 18 חודשים = **₪6,282**
- **CAC מקסימלי:** ₪1,500 (יחס LTV:CAC = 4:1)
- **Churn צפוי:** 5-8% חודשי (נורמלי ל-SMB SaaS)

### על השוק
- **שוק יעד:** ארגונים וחברות בישראל (2-200 עובדים)
- **TAM (Total Addressable Market):** ~500K עסקים בישראל
- **SAM (Serviceable Available Market):** ~100K (עסקים דיגיטליים)
- **SOM (Serviceable Obtainable Market):** ~10K (שנה 1)

## 🚀 ערוצי רכישה (Acquisition Channels)

### אורגני (0 תקציב)
1. **LinkedIn Organic** — פוסטים יומיים, מאמרים
2. **YouTube SEO** — סרטוני הדרכה ארוכים
3. **קבוצות פייסבוק** — עזרה אמיתית, לא ספאם
4. **SEO** — בלוג, מדריכים
5. **Product Hunt / Reddit** — השקות

### Paid (תקציב מועט)
1. **Google Ads** — Search (כוונה גבוהה)
2. **Meta Ads** — רימרקטינג + Lookalike
3. **LinkedIn Ads** — B2B מכוון

### Paid (תקציב מלא)
1. **Google Ads** — Search + Display + YouTube
2. **Meta Ads** — Awareness + Consideration + Conversion
3. **LinkedIn Ads** — Sponsored Content + InMail
4. **Outbound** — Cold Email + LinkedIn Outreach
5. **אינפלואנסרים** — מיקרו-אינפלואנסרים B2B
