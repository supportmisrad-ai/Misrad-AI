## A/B Testing Framework

### מה לבדוק

#### Creative Testing
1. **Image vs Video**
2. **Short vs Long Copy**
3. **Different Headlines**
4. **CTA Buttons:** "התחל חינם" vs "למד עוד" vs "בקש דמו"

#### Audience Testing
1. **Age Brackets:** 25-35 vs 35-45 vs 45-55
2. **Interests:** Broad vs Narrow
3. **Lookalike %:** 1% vs 2% vs 3%

#### Landing Page Testing
1. **Short Form vs Long Form**
2. **Video Demo vs No Video**
3. **Social Proof Placement**

### תהליך
1. Run 2 variants simultaneously
2. Wait for 1,000 impressions minimum
3. Winner = Lower CPA + Higher CR
4. Kill loser, launch new variant
