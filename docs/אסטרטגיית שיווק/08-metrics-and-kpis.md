## 📊 Funnel Metrics

### Top of Funnel (TOFU)

```
Impressions: 100,000
    ↓ (2% CTR)
Clicks: 2,000
    ↓ (50% Bounce)
Landing Page Views: 1,000
```

### Middle of Funnel (MOFU)

```
Landing Page Views: 1,000
    ↓ (15% Conversion)
Trial Sign-ups: 150
    ↓ (60% Activation)
Active Trials: 90
```

### Bottom of Funnel (BOFU)

```
Active Trials: 90
    ↓ (25% Conversion)
Paid Customers: 22-23
```

**Overall Conversion:** 100,000 impressions → 22 customers = **0.022%**

## 💰 Unit Economics

### Customer Lifetime Value (LTV)

```
ARPU: ₪349/חודש
Avg Lifetime: 18 חודשים
LTV = ₪349 × 18 = ₪6,282
```

### Customer Acquisition Cost (CAC)

```
Monthly Ad Spend: ₪35,000
Monthly Customers: 100
CAC = ₪35,000 / 100 = ₪350
```

### LTV:CAC Ratio

```
₪6,282 / ₪350 = 17.9:1 ✅

Target: >3:1
Healthy: >3:1
Great: >5:1
Excellent: >10:1
```

### Payback Period

```
CAC / ARPU = ₪350 / ₪349 = 1 חודש ✅

Target: <12 חודשים
Great: <6 חודשים
Excellent: <3 חודשים
```

## 📈 Growth Tracking

### Month-over-Month (MoM) Growth

```
MRR Month 2: ₪20K
MRR Month 1: ₪10K

MoM Growth = (₪20K - ₪10K) / ₪10K × 100% = 100% 🚀
```

**Target MoM Growth:**
- חודשים 1-3: 50-100%
- חודשים 4-9: 30-50%
- חודשים 10-12: 20-30%
