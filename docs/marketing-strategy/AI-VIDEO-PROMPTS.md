# פרומפטים ויזואליים — סרטון "האם ה-AI יחסל לך את העבודה?"
## 29 קליפים × 8 שניות = 232 שניות (~3:52)
## כל פרומפט מיועד ליצירת קליפ AI בן 8 שניות

---

### הנחיות כלליות לכל הפרומפטים:

**סגנון בסיס (להוסיף בסוף כל פרומפט):**
```
Cinematic 4K, 24fps, anamorphic lens, shallow depth of field, 
subtle Dutch angle 2-5°, film grain overlay, no flat colors, 
diffused colored shadows, optical bloom on light sources, 
camera with natural inertia movement.
```

**אסור בכל הקליפים:**
- אייקוני מוחות / רשתות נוירונים / קווים זוהרים
- צבעים שטוחים (Flat) — תמיד grain + texture
- Typewriter effect
- כרטיסים נופלים / gravity drops
- Canva-level graphics

## הנחיות המשכיות בין קליפים

### פלטת צבעים לכל הסרטון:
| פרק | טמפרטורת צבע | מצב רוח |
|------|-------------|---------|
| Hook (1-3) | קר-כחלחל + אור מסך | חרדה שקטה |
| "מחקר" (4-6) | ניטרלי → חם | חשיפה, ספקנות |
| מציאות (7-10) | ניטרלי מאוזן | עובדתי, רגוע |
| בעיה/לא (11-14) | קר (בעיה) → חם (לא בעיה) | קונטרסט |
| היסטוריה (15-17) | ספיה → מודרני → כהה | זמן, תנועה |
| AI לא יכול (18-20) | קר-כסוף → חם-סדנה | מכונה vs אדם |
| כסף (21-23) | כהה + זהב | עלות, פער |
| הימור (24-26) | טבעי → ערפל | אי-ודאות יפה |
| סגירה (27-29) | זריחה זהובה → אור מלא | אמונה, קדימה |

### רצף תנועת מצלמה:
- קליפים 1-6: תנועות **פנימה** (push in, macro) — חקירה
- קליפים 7-14: תנועות **צידה** (tracking, orbiting) — סקירה
- קליפים 15-17: תנועות **קדימה בזמן** (through time) — היסטוריה
- קליפים 18-20: תנועות **drift** (צפות) — מופשט → מוחשי
- קליפים 21-23: תנועות **מלמעלה למטה** (aerial, descent) — משקל, עלות
- קליפים 24-26: **dolly forward רציף** — אותו כיוון לאורך 3 קליפים!
- קליפים 27-29: **מסטטי → קדימה** — מהתבוננות לתנועה

### טיפ ליצירת המשכיות:
בסוף כל פרומפט אפשר להוסיף:
```
End frame: [תיאור הפריים האחרון] — next clip starts from similar composition.
```
כך שהקליפ הבא מתחיל מנקודה דומה ויוצר flow.
