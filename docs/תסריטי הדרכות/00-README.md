## 🎯 מבנה כל תסריט

כל תסריט בנוי בפורמט אחיד:

```
## מטא-דאטה
- אורך משוער
- קהל יעד
- מטרת הסרטון
- מסרים מרכזיים

## פתיחה (Hook)
- 10-15 שניות ראשונות שתופסות תשומת לב

## גוף הסרטון
- סצנות ממוספרות
- כל סצנה כוללת:
  - [מה מראים על המסך]
  - [טקסט דיבור / Voice-over]
  - [הערות ביים]

## סיום + CTA
- סיכום
- קריאה לפעולה
```

## 💡 הנחיות צילום כלליות

### טון ושפה
- **עברית טבעית** — לא שיווקית-זולה, לא טכנית מדי
- **ישיר ובגובה העיניים** — כאילו מסביר לחבר
- **כנות** — אומרים מה יש ומה אין

### ויזואליות
- **הקלטת מסך** — Screen recording של המערכת בפועל
- **Zoom In** — על כפתורים ואזורים חשובים
- **Annotations** — חיצים, הדגשות, מספרים
- **Transitions** — מעברים חלקים בין מסכים

### Voice-Over
- קול רגוע ובטוח
- קצב: ~130-150 מילים/דקה (לא מהר מדי)
- הפסקות קצרות בין נקודות מפתח
