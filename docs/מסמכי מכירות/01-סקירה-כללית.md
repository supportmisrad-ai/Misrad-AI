# 🎯 סקירה כללית — Misrad-AI

## מהי המערכת?

**MISRAD AI** — מערכת AI שמנהלת את הארגון במקומך — מנתחת, מחזה, וממליצה.

| מאפיין | ערך |
|--------|------|
| **סוג** | SaaS Multi-Tenant |
| **מודולים** | 6 עצמאיים |
| **AI** | Google Gemini 2.0 |
| **מחיר** | ₪149–₪499/חודש |
| **Stack** | Next.js 16, React 19, TypeScript |

---

## שם המערכת

**MISRAD AI** — זה השם היחיד. אין שמות נוספים.

> **Nexus** הוא שם של מודול מרכזי (ניהול צוות, משימות, לקוחות) — לא שם המערכת.

## תפיסת Modular OS

המערכת = **מערכת AI שמנהלת את הארגון** — כל מודול = תוכנה נפרדת.

```
Misrad-AI OS
├── 📊 Nexus   (ניהול צוות)     ← ✅ / ❌
├── 💼 System  (מכירות)         ← ✅ / ❌
├── 📱 Social  (שיווק)          ← ✅ / ❌
├── 💰 Finance (כספים)          ← ✅ / ❌
├── 👥 Client  (לקוחות)         ← ✅ / ❌
└── 🔧 Operations (תפעול)      ← ✅ / ❌

תשלם רק על מה שמופעל!
```
