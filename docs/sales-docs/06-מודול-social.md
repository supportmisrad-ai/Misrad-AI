## מסכים ופיצ'רים

### 1. לוח שנה עברי ⭐ (ייחודי!)

| פיצ'ר | תיאור |
|--------|--------|
| **תאריכים עבריים** | כ"ד שבט, ט"ו באב — ליד הלועזי |
| **חגים** | סימון אוטומטי של חגים, צומות, ימים מיוחדים |
| **הגנת שבת** | `is_shabbat_protected` — מניעת פרסום אוטומטית בשבת |
| **תצוגה חודשית** | Calendar עם פוסטים מתוזמנים |

> **למה זה חשוב?** אף מתחרה (HubSpot, Hootsuite, Buffer) לא מציע לוח שנה עברי.  
> עסקים ישראליים דתיים/מסורתיים — זה פיצ'ר קריטי.

> ⚠️ **למי שלא שומר שבת:** הגנת שבת היא בחירה אידיאולוגית של המפתח. אם זה לא מתאים לך — יש Monday.com, Fireberry, הכוורת ועוד מתחרים מצויינים שפתוחים 24/7. אף אחד לא כופה עליך לקנות.

### 2. Content Machine — יצירת תוכן AI ⭐

| פיצ'ר | תיאור |
|--------|--------|
| **יצירה מהירה** | הזן נושא → 3 וריאציות |
| **התאמה** | כל וריאציה מותאמת לפלטפורמה (Facebook / Instagram / LinkedIn) |
| **Hashtags חכמים** ⭐ NEW | AI ממליץ hashtags מותאמים לפלטפורמה (Instagram: 5-10, LinkedIn: 3-5, Facebook: 1-2) |
| **שעות פרסום מיטביות** ⭐ NEW | AI אומר מתי לפרסם לפי engagement ("רביעי 11:00 - 85%") |
| **הצעות תמונה** | תיאור visual מ-AI |
| **עברית מלאה** | קופי בעברית טבעית |
| **שפות** | עברית + אנגלית |

**דוגמה:**
```
Input: "פוסט על מבצע סוף שנה 30% הנחה"

Output:
├── Facebook: 
│   Content: "🎉 מבצע סוף שנה! 30% הנחה על כל המוצרים..."
│   Hashtags: ["#מבצע", "#הנחה"]
│   Best time: "רביעי 11:00 (engagement: 85%)"
│
├── Instagram: 
│   Content: "30% OFF 🔥"
│   Hashtags: ["#מבצע", "#הנחה", "#סוףשנה", "#sale", "#shopping", "#ישראל", "#טכנולוגיה"]
│   Best time: "שלישי 12:00 (reach: 88%)"
│
└── LinkedIn: 
    Content: "אנו שמחים להכריז על מבצע מיוחד..."
    Hashtags: ["#BusinessDeal", "#B2B", "#עסקים"]
    Best time: "רביעי 12:00 (professional engagement: 87%)"
```

### 3. תזמון פוסטים

| פיצ'ר | תיאור |
|--------|--------|
| **תזמון** | בחר תאריך + שעה |
| **פלטפורמות** | Facebook, Instagram, LinkedIn, Twitter, TikTok |
| **סטטוסים** | Draft → Scheduled → Published / Failed |
| **חזרתיות** | יומי / שבועי / חודשי |
| **Late Detection** | `is_late` — סימון פוסטים שפורסמו באיחור |

### 4. Campaigns — קמפיינים

| פיצ'ר | תיאור |
|--------|--------|
| **ניהול קמפיין** | שם, תאריכים, מטרה |
| **פוסטים מקושרים** | קבוצת פוסטים תחת קמפיין |
| **מעקב** | כמה פורסם / כמה נשאר |

### 5. בנק רעיונות

| פיצ'ר | תיאור |
|--------|--------|
| **שמירת רעיון** | טקסט חופשי |
| **תיוג** | קטגוריות |
| **הפיכה לפוסט** | לחץ → הפוך לפוסט מתוזמן |

### 6. Conversations — Inbox

| פיצ'ר | תיאור |
|--------|--------|

### 7. ניהול מרובה לקוחות (סוכנויות)

| פיצ'ר | תיאור |
|--------|--------|
| **לקוחות** | ניהול מספר לקוחות שיווק |

## תהליכי עבודה

### תהליך: יצירת תוכן שבועי
```
1. פתח Calendar → ראה מה חסר השבוע
2. Content Machine → יצירת 5 פוסטים AI
3. ערוך / אשר כל פוסט
4. תזמן → בחר תאריך ושעה
5. פרסום אוטומטי (הגנת שבת פעילה)
```

### תהליך: סוכנות עם 10 לקוחות
```
1. בחר לקוח מהרשימה
2. ראה Calendar שלו
3. צור תוכן (AI או ידני)
4. לקוח מאשר (דרך Portal - מודול Client)
5. תזמן ופרסם
```
