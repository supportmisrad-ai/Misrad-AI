# 💾 גיבוי מלא של הפרויקט לענן

## 🎯 למה צריך גיבוי מלא?

יש **2 סוגים** של דברים לגבות:

### 1. **הקוד (Code)**
- ✅ **Git כבר מגבה** - GitHub/GitLab
- כל קומיט נשמר בענן
- אם המחשב נשרף - הקוד בטוח

### 2. **הדאטאבייס (Data)**
- ⚠️ **לא ב-Git!** (ולא צריך להיות)
- Supabase מגבה יומית (7 ימים)
- הגיבויים המקומיים שלך (`backups/`)

**הבעיה:** אם הדיסק נשרף, גם הגיבויים המקומיים נעלמים! 🔥

---

## 🛡️ **הפתרון: גיבוי מלא לענן**

### מה זה כולל:
- ✅ כל קבצי הקוד (ל-Git)
- ✅ גיבויי הדאטאבייס (`backups/*.json`)
- ✅ קבצי הגדרות (`.env.local` - אופציונלי)
- ✅ כל קובץ אחר שחשוב

### לאן לגבות:
1. **OneDrive** (אם יש לך Windows)
2. **Google Drive**
3. **Dropbox**
4. **דיסק חיצוני** (USB/SSD)

---

## ⚙️ הגדרה - 2 שלבים

### שלב 1: בחר מיקום גיבוי

**אופציה א: OneDrive (מומלץ למשתמשי Windows)**
```powershell
$backupLocation = "$env:OneDrive\Backups\Misrad-AI"
```

**אופציה ב: Google Drive**
```powershell
$backupLocation = "G:\My Drive\Backups\Misrad-AI"
```

**אופציה ג: דיסק חיצוני**
```powershell
$backupLocation = "D:\Backups\Misrad-AI"  # שנה D:\ לאות הדיסק שלך
```

### שלב 2: הרץ הגדרה אוטומטית

**פתח PowerShell כ-Administrator:**
```powershell
cd C:\Projects\Misrad-AI
.\scripts\setup-weekly-full-backup.ps1
```

**הסקריפט ישאל אותך:**
- "להמשיך עם המיקום הזה? (y/n)"
- אפשר לשנות אם רוצה

**זהו!** המשימה תרוץ **כל יום ראשון ב-2:00 בלילה** ✅

---

## 🧪 בדיקה ידנית

רוצה לבדוק שזה עובד? הרץ עכשיו:

```powershell
.\scripts\full-project-backup.ps1 -BackupDestination "$env:OneDrive\Backups\Misrad-AI"
```

**Google Drive:**
```powershell
.\scripts\full-project-backup.ps1 -BackupDestination "G:\My Drive\Backups\Misrad-AI"
```

**דיסק חיצוני:**
```powershell
.\scripts\full-project-backup.ps1 -BackupDestination "D:\Backups\Misrad-AI"
```

## 📦 מה נשמר בגיבוי?

### כלול:
- ✅ כל קבצי הקוד (`.js`, `.ts`, `.tsx`, וכו')
- ✅ `backups/` - גיבויי הדאטאבייס
- ✅ `prisma/` - הסכמה
- ✅ `package.json`, `.env.example`
- ✅ כל קובץ חשוב אחר

### לא כלול (כדי לחסוך מקום):
- ❌ `node_modules/` - אפשר להתקין מחדש
- ❌ `.next/` - build artifacts
- ❌ `dist/`, `build/`
- ❌ `.git/` - יש ב-GitHub

**גודל משוער:** ~50-100 MB (בלי `node_modules`)

## 🔄 מה קורה אוטומטית?

**כל יום ראשון ב-2:00 בלילה:**

1. 💾 גיבוי דאטאבייס חדש (`npm run db:backup`)
2. 📦 העתקת כל הפרויקט למיקום הגיבוי
3. 📋 יצירת קובץ `backup-info.json` עם מטה-דאטה
4. 🧹 מחיקת גיבויים ישנים (שומר 5 אחרונים)

**אם המחשב כבוי:** ירוץ ברגע שתדליק ✅

## 🚨 שחזור מגיבוי מלא

אם המחשב נשרף/נגנב/נשבר:

### 1. התקן מחדש:
- Node.js
- Git

### 2. העתק את הגיבוי האחרון:
```powershell
Copy-Item "$env:OneDrive\Backups\Misrad-AI\Misrad-AI_Full_2026-02-04_14-30-00" -Destination "C:\Projects\Misrad-AI" -Recurse
```

### 3. התקן תלויות:
```powershell
cd C:\Projects\Misrad-AI
npm install
```

### 4. שחזר דאטאבייס:
```powershell
npm run db:restore
```

### 5. הפעל:
```powershell
npm run dev
```
