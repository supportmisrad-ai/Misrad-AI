## 🎯 מהות המערכת

### סוג הבוט
בוט וואטסאפ אוטומטי ל-MISRAD AI המבוסס על מערכת כללים (Rule-Based)

### תכלית
- קבלת פניות מלקוחות פוטנציאליים בוואטסאפ
- מענה אוטומטי לפי כללים מוגדרים מראש
- איסוף מידע על הצרכים של הלקוח
- שליחת לידים למערכת MISRAD AI
- תמיכה ב-4 סוגי לידים: lead | signup | demo | support

---

## 📊 מבנה הקובץ Rules_5.rules

### פורמט
קובץ JSON בודד (single line) המכיל מערך של כללי בוט.

**מבנה כללי:**
```json
[
  {
    "Rule": "כותרת הכלל##מילת_מפתח_1##מילת_מפתח_2",
    "response": "תשובת הבוט ללקוח",
    "extra": {
      "Active": true,
      "Webhook": "https://misrad-ai.com/api/webhooks/blaster?type=lead",
      "SendWebhook": true,
      "GoToRule": 1,
      "WaitFor": 0,
      "Delay": 0
    }
  }
]
```

## 🔌 Webhooks מוגדרים

### Endpoint מרכזי
```
POST https://misrad-ai.com/api/webhooks/blaster
```

### סוגי Webhooks פעילים

| סוג | תיאור | כללים מקושרים |
|-----|-------|---------------|
| **lead** | ליד חדש - התעניינות ראשונית | 1, 20, 24-28 |
| **signup** | לקוח סיים הרשמה | 37 |
| **demo** | בקשת הדגמה/פגישה | 43 |
| **support** | פניה לתמיכה | 63 |

### 12 כללים עם Webhooks פעילים

1. **כלל 1** - פתיחה/שלום → type=lead
2. **כלל 20** - חבילות ומחירים → type=lead
3. **כלל 24** - "לידים נופלים בין הכיסאות" → type=lead
4. **כלל 25** - "אין שליטה על הצוות" → type=lead
5. **כלל 26** - "השיווק לא מניב" → type=lead
6. **כלל 27** - "פרויקטים חורגים מתקציב" → type=lead
7. **כלל 28** - "אני רוצה הכל" → type=lead
8. **כלל 37** - "סיימו את ההרשמה" → type=signup
9. **כלל 43** - בוקר/צהריים/ערב → type=demo
10. **כלל 63** - נציג אנושי → type=support
11. **כלל %%** - תמיכה טכנית → type=support
12. **כלל %%** - הסרה/סטופ → type=support

## 📝 תוכן ענייני הכללים

### עדכון אחרון (מרץ 2025)

**שינויים מרכזיים:**
- חבילת "Solo" → "נקסוס בלבד" ₪149
- הסרת אפשרות בחירת מודולים בודדים
- 4 חבילות → "נקסוס בלבד" + חבילות משולבות
- כל החבילות (למעט נקסוס בלבד) כוללות 5 משתמשים

### מודולים שמוצגים בבוט
- **Nexus** (ניהול צוות, משימות, לקוחות) - מודול יחיד שניתן לרכישה נפרדת
- **System** - מגיע בחינם עם כל החבילות
- **Social** - מודול שיווק וניהול תוכן
- **Client** - ניהול לקוחות ופניות
- **Operations** - ניהול פרויקטים ותפעול

**לא מוצג:** Finance (מודול חינמי שמגיע עם כל החבילות)

## 🗂️ טבלאות במסד הנתונים

### botLead
- שמירת לידים חדשים מהבוט
- כולל: מספר טלפון, שם, מייל, ענף, חבילה נבחרת
- סטטוס: new → contacted → qualified → converted

### botConversation
- לוג שיחות מלא עם הבוט
- מיפוי כל אינטראקציה
- שימוש לניתוח ושיפור הבוט

## 🔄 תהליך אינטגרציה
