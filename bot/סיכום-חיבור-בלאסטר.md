## 🚀 השלבים הבאים

### 1. גיבוי (לפני שינוי)
```powershell
# בתיקיית WAAM-It Blaster
ren BotApi.json BotApi-backup.json
ren BotApi-MISRAD-AI.json BotApi.json
```

### 2. עדכון Secret
```json
// ב-BotApi.json
"WebhookSecret": "הסוד_האמיתי_מ_env"
```

### 3. הפעלה מחדש
- סגור את הבלאסטר לגמרי
- הפעל מחדש

### 4. בדיקה
```powershell
# מ-Terminal בפרויקט Misrad-AI
node scripts/test-blaster-webhook.js
# או
.\scripts\test-blaster-webhook.ps1
```

## ⚡ מהיר - רק השורות שצריך להעתיק

### ב-BotApi.json החדש:
```json
{
  "Sheets": {
    "LogSheet": {
      "AccessType": 1,
      "Id": "https://misrad-ai.com/api/webhooks/blaster",
      "Sheet": "bot_leads",
      "Headers": {
        "phone": 1, "name": 2, "business": 3, "industry": 4,
        "org_size": 5, "pain_point": 6, "selected_plan": 7,
        "email": 8, "message": 9, "rule_id": 10, "type": 11
      }
    }
  },
  "MISRAD_AI": {
    "WebhookSecret": "YOUR_SECRET_HERE"
  }
}
```

### ב-.env של המערכת:
```env
BLASTER_WEBHOOK_SECRET=YOUR_SECRET_HERE
```

---

## 🎯 מה יקרה אחרי ההתקנה

1. **לידים** ישמרו ישירות בטבלת `botLead`
2. **שיחות** יתועדו בטבלת `botConversation`
3. **סטטוס** יתעדכן אוטומטית (new/trial/demo_booked)
4. **ציון** יחושב אוטומטית לפי אינטראקציות
5. **גוגל שיטס** לא יהיה רלוונטי יותר

  -H "x-webhook-secret: YOUR_SECRET" \
