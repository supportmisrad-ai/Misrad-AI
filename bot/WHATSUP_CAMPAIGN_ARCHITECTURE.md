## 2. הפתרון האופטימלי

### גישה נבחרת: **לינק אישי לכל שותף**

**למה?**
- ✅ חוויית משתמש פשוטה - הלקוח לא צריך להזין קוד
- ✅ מעקב אוטומטי - השותף מקבל עמלה בלי פעולה ידנית
- ✅ הנחה אוטומטית - 50% לחצי שנה מופעלת לבד
- ✅ שקיפות מלאה - הכל מתועד ב-DB

### זרימת המשתמש

```
שותף מקבל לינק: https://misrad-ai.com/signup?ref=DANIEL123
                                    ↓
לקוח נכנס → Clerk sign-up עם ?ref=DANIEL123 ב-URL
                                    ↓
הרשמה → שמירת ref ב-metadata של Clerk
                                    ↓
יצירת ארגון → partnerId = Partner.id (מהקוד DANIEL123)
                                    ↓
הנחה אוטומטית: 50% לחצי שנה (דרך GlobalPromotion או קופון וירטואלי)
                                    ↓
שותף מקבל: 10% עמלה חודשית + בונוס 150₪ אחרי 3 חודשים
```

## 4. מודל עמלות

### לקוח חדש (דרך שותף)
- **הנחה:** 50% לחצי שנה (6 חודשים)
- **תנאי:** חייב להירשם דרך לינק אישי או לציין שותף בבוט

### שותף (Affiliate)
- **עמלה:** 10% מהכנסה חודשית של הלקוח (עד שנה)
- **בונוס:** 150₪ נקי על כל לקוח שמשלם מעל 3 חודשים
- **תשלום:** חודשי או שנתי (לבחירת השותף)

### דוגמה חישובית
```
לקוח נרשם ל-The Empire (499₪/חודש)
חודש 1-6: 249.5₪ (50% הנחה) → שותף מקבל 24.95₪/חודש
חודש 7-12: 499₪ → שותף מקבל 49.9₪/חודש
חודש 4: בונוס 150₪ (עבר 3 חודשים)

סה"כ לשותף: (24.95×6) + (49.9×6) + 150 = 149.7 + 299.4 + 150 = 599.1₪
```
