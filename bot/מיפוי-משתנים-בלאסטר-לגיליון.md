# מיפוי משתנים - בלאסטר לגוגל שיטס

## 🎯 משתנים לשליחה מהבלאסטר

### משתנים חובה (מומלץ לשלוח תמיד):
| משתנה בלאסטר | עמודה בגיליון | הסבר |
|--------------|---------------|------|
| `{contact}` | Contact | מספר טלפון |
| `{WhatsName}` | WhatsName | שם בוואטסאפ |
| `{firstname}` | First_Name | שם פרטי |
| `{business}` | Business_Name | שם העסק |

### משתנים חשובים (מומלץ):
| משתנה בלאסטר | עמודה בגיליון | הסבר |
|--------------|---------------|------|
| `{lastname}` | Last_Name | שם משפחה |
| `{email}` | Email | אימייל |
| `{industry}` | Industry | תחום עיסוק |
| `{orgsize}` | Org_Size | גודל ארגון |
| `{message}` | Message | תוכן ההודעה |
| `{source}` | Source | מקור הליד |
| `{campaign}` | Campaign | קמפיין |
| `{painpoint}` | Pain_Point | נקודת כאב |
| `{selectedplan}` | Selected_Plan | חבילה שנבחרה |

### משתנים נוספים (אופציונלי):
| משתנה בלאסטר | עמודה בגיליון | הסבר |
|--------------|---------------|------|
| `{date}` | Date_Created | תאריך |
| `{time}` | Time_Created | שעה |
| `{ruleid}` | Rule_ID | מזהה כלל |

## 📋 הטמעה בבלאסטר

### שיטה 1: URL עם פרמטרים (פשוטה)
```
https://script.google.com/macros/s/XXXX/exec?contact={contact}&WhatsName={WhatsName}&firstname={firstname}&lastname={lastname}&business={business}&industry={industry}&orgsize={orgsize}&message={message}&source=blaster&campaign={campaign}
```

### שיטה 2: JSON Body (מתקדם)
```json
{
  "contact": "{contact}",
  "WhatsName": "{WhatsName}",
  "firstname": "{firstname}",
  "lastname": "{lastname}",
  "business": "{business}",
  "industry": "{industry}",
  "orgsize": "{orgsize}",
  "message": "{message}",
  "source": "blaster",
  "campaign": "{campaign}",
  "painpoint": "{painpoint}",
  "selectedplan": "{selectedplan}",
  "email": "{email}"
}
```

## 🔧 הגדרות בבלאסטר

### שלב 1: פתח הגדרות API
1. **פתח** את תוכנת בלאסטר
2. **לך ללשונית** "API" או "Integrations"
3. **בחר** "שליחת הודעות" וגם "לוג הודעות" (גם וגם)

### שלב 2: הכנס URL
- **הדבק** את ה-Web App URL שקיבלת מ-Google Apps Script

### שלב 3: בחר משתנים
סמן V על המשתנים שאתה רוצה לשלוח:
- ☑️ contact
- ☑️ WhatsName  
- ☑️ firstname
- ☑️ business
- ☑️ industry (אם יש)
- ☑️ orgsize (אם יש)
- ☑️ message (אם יש)

## 🆘 פתרון בעיות

**"קישור לא תקין"**
→ בדוק שה-URL נכון ושה-Web App מוגדר כ"Anyone"

**ליד לא נכנס לגיליון**
→ בדוק ב-Logs של Google Apps Script (View → Logs)

**משתנה לא מופיע**
→ ודא שכתבת את השם המדויק (case sensitive)
