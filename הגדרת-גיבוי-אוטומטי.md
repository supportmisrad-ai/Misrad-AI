# 🤖 הגדרת גיבוי אוטומטי יומי

## 🎯 מטרה
הגיבויים ירוצו **אוטומטית כל יום** על המחשב שלך (לא תלוי באינטרנט/Git).

---

## ⚙️ אופציה 1: Windows Task Scheduler (מומלץ!)

### שלב 1: הרץ סקריפט ההגדרה

**פתח PowerShell כ-Administrator:**
1. לחץ Start
2. חפש "PowerShell"
3. לחץ ימני → "Run as Administrator"

**הרץ:**
```powershell
cd C:\Projects\Misrad-AI
.\scripts\setup-daily-backup.ps1
```

### מה זה עושה?
- ✅ יוצר משימה מתוזמנת ב-Windows
- ✅ רץ כל יום ב-**9:00 בבוקר**
- ✅ מריץ `npm run db:backup` אוטומטית
- ✅ הגיבויים נשמרים ב-`backups/` אצלך

### לבדוק שזה עובד:
1. פתח **Task Scheduler** (חפש בStart)
2. לך ל-**Task Scheduler Library**
3. חפש: **"Misrad-AI Daily Backup"**
4. לחץ ימני → **Run** (בדיקה ידנית)
5. בדוק ש-`backups/` קיבל קובץ חדש

### לשנות שעה:
1. Task Scheduler → מצא את המשימה
2. לחץ ימני → **Properties**
3. לשונית **Triggers** → **Edit**
4. שנה את השעה → **OK**

## ⚙️ אופציה 2: GitHub Actions (לא מומלץ לגיבויים מקומיים)

**למה לא מומלץ:**
- ❌ רץ על שרתי GitHub (לא אצלך)
- ❌ צריך DATABASE_URL ב-GitHub Secrets
- ❌ הגיבוי נשמר ב-GitHub (לא במחשב)
- ⚠️ GitHub רואה את הנתונים שלך

**אם בכל זאת רוצה:**
<details>
<summary>לחץ להרחבה</summary>

קובץ: `.github/workflows/daily-backup.yml`

```yaml
name: Daily Database Backup

on:
  schedule:
    - cron: '0 6 * * *'  # כל יום ב-6:00 UTC (9:00 בישראל)
  workflow_dispatch:  # אפשר להריץ ידנית

jobs:
  backup:
    runs-on: ubuntu-latest
    
    steps:
    - uses: actions/checkout@v3
    
    - name: Setup Node.js
      uses: actions/setup-node@v3
      with:
        node-version: '18'
    
    - name: Install dependencies
      run: npm install
    
    - name: Run backup
      env:
        DATABASE_URL: ${{ secrets.DATABASE_URL }}
      run: npm run db:backup
    
    - name: Upload backup
      uses: actions/upload-artifact@v3
      with:
        name: database-backup-${{ github.run_number }}
        path: backups/
        retention-days: 30
```

**צריך להוסיף Secret:**
1. GitHub Repo → Settings → Secrets → New repository secret
2. Name: `DATABASE_URL`
3. Value: [ה-DATABASE_URL שלך]

</details>

## 📅 מתי הגיבויים ירוצו?

### Windows Task Scheduler:
- **כל יום ב-9:00 בבוקר** (ברירת מחדל)
- אפשר לשנות לכל שעה שתרצה
- רץ רק כשהמחשב **דלוק**

### טיפים:
- ✅ אם המחשב כבוי ב-9:00 - ירוץ ברגע שתדליק
- ✅ עובד גם על סוללה (מוגדר)
- ✅ לא תלוי באינטרנט (אחרי ההתחברות לDB)

## 🔍 איך לבדוק שזה עובד?

### בדיקה 1: הרץ ידנית
```powershell
npm run db:backup
```
אמור ליצור קובץ ב-`backups/`

### בדיקה 2: בדוק Task Scheduler
```powershell
Get-ScheduledTask -TaskName "Misrad-AI Daily Backup"
```
אמור להראות את המשימה

### בדיקה 3: הרץ את המשימה ידנית
Task Scheduler → לחץ ימני → Run

### בדיקה 4: בדוק את ההיסטוריה
Task Scheduler → History (בתחתית)

## 🛠️ פתרון בעיות

### "Access Denied" בהרצת הסקריפט
```powershell
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```

### המשימה לא רצה
1. בדוק ב-Task Scheduler → History
2. ראה את השגיאה
3. וודא ש-Node.js בנתיב (PATH)

### הגיבוי לא נוצר
```powershell
# הרץ ידנית לראות שגיאות
node C:\Projects\Misrad-AI\scripts\auto-backup.js
```
