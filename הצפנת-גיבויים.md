# 🔐 גיבויים מוצפנים

## 🎯 למה צריך הצפנה?

אם אתה מגבה ל:
- ☁️ OneDrive / Google Drive
- 📧 מייל
- 💾 דיסק חיצוני משותף

**צריך הצפנה!** כדי שאף אחד לא יוכל לקרוא את הנתונים.

---

## ⚙️ הגדרה - פעם אחת

### 1. התקן 7-Zip (חינמי)
https://www.7-zip.org/download.html

הורד את הגרסה ל-Windows x64 והתקן.

### 2. הרץ גיבוי מוצפן ראשון
```powershell
cd C:\Projects\Misrad-AI
.\scripts\encrypted-backup.ps1
```

**מה קורה:**
1. יוצר גיבוי מלא של הפרויקט
2. **מצפין אותו** עם AES-256 (הכי חזק)
3. יוצר **סיסמה אוטומטית** ושומר ב-`.backup-password.txt`
4. שומר את הקובץ המוצפן ב-OneDrive (או מיקום אחר)

## 🔑 הסיסמה

הסקריפט יוצר קובץ: `.backup-password.txt`

**חשוב:**
- ✅ הקובץ **לא עולה ל-Git** (ב-.gitignore)
- ⚠️ **שמור עותק נוסף** במקום בטוח!
- ⚠️ **בלי הסיסמה - לא תוכל לשחזר!**

**איפה לשמור עותק:**
- 📧 שלח לעצמך למייל
- 📱 שמור בטלפון (Notes/Keep)
- 📝 רשום על נייר (ישן אבל עובד)

## 🚀 שימוש

### גיבוי מוצפן ידני:
```powershell
.\scripts\encrypted-backup.ps1
```

### גיבוי למיקום מסוים:
```powershell
.\scripts\encrypted-backup.ps1 -BackupDestination "D:\Secure-Backups"
```

### עם סיסמה מותאמת אישית:
```powershell
.\scripts\encrypted-backup.ps1 -Password "הסיסמה-החזקה-שלי-123!"
```

## 🔓 שחזור מגיבוי מוצפן

### אופציה 1: דרך 7-Zip (GUI)
1. פתח את הקובץ `Misrad-AI_Encrypted_2026-02-04_14-30-00.7z`
2. 7-Zip יבקש סיסמה
3. הזן מ-`.backup-password.txt`
4. חלץ לתיקייה

### אופציה 2: שורת פקודה
```powershell
# קרא את הסיסמה
$password = Get-Content .backup-password.txt -Raw

# חלץ
& "C:\Program Files\7-Zip\7z.exe" x "Misrad-AI_Encrypted_2026-02-04_14-30-00.7z" -p"$password" -o"C:\Restored"
```

## 🤖 גיבוי מוצפן אוטומטי

אפשר להגדיר Task Scheduler:

```powershell
# יצירת משימה שבועית
$action = New-ScheduledTaskAction `
    -Execute "powershell.exe" `
    -Argument "-NoProfile -ExecutionPolicy Bypass -File `"C:\Projects\Misrad-AI\scripts\encrypted-backup.ps1`"" `
    -WorkingDirectory "C:\Projects\Misrad-AI"

$trigger = New-ScheduledTaskTrigger -Weekly -DaysOfWeek Sunday -At 3:00AM

Register-ScheduledTask `
    -TaskName "Misrad-AI Encrypted Weekly Backup" `
    -Action $action `
    -Trigger $trigger `
    -User $env:USERNAME `
    -RunLevel Highest
```
